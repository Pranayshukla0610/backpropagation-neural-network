# backpropagation-neural-network
A Deep Learning implementation of a Backpropagation Neural Network (BPNN) for supervised learning tasks. This project demonstrates forward propagation, backpropagation, gradient descent optimization, and weight updates from scratch and using modern deep learning libraries.

## Features

- Forward Propagation
- Backpropagation Algorithm
- Gradient Descent Optimization
- Multi-Layer Perceptron (MLP)
- Custom Activation Functions
- Model Training & Evaluation
- Performance Visualization
- Easy-to-Understand Implementation

## Tech Stack

- Python
- NumPy
- TensorFlow / PyTorch
- Matplotlib
- Scikit-learn

## Applications

- Classification
- Regression
- Pattern Recognition
- Image Classification
- Predictive Analytics
